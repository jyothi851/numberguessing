Excited to share the completion of **Task 2** of my Python internship at **Reciprocity**.

### 📌 Project Overview

Developed a **Number Guessing Game** in Python where the computer randomly selects a number between **1 and 100**, and the user keeps guessing until the correct number is found.

### ✅ Features

* Generates a random number between 1 and 100 using Python's `random` module
* Accepts user guesses through the command line
* Provides hints:

  * 📈 "Too high" if the guess is greater than the target
  * 📉 "Too low" if the guess is smaller than the target
* Continues until the correct number is guessed
* 🎉 Displays the total number of attempts after a successful guess

### 🛠️ Technologies Used

* Python
* `random` Module
* Loops (`while`)
* Conditional Statements (`if-elif-else`)
* User Input Handling

This project helped me strengthen my understanding of **random number generation, loops, conditional logic, and interactive command-line applications**.
